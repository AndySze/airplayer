AirPlayer
================================================================================

Command-line AirPlay video client

---


Show my AirPlay devices

```sh
$ airplayer devices
Apple TV: 10.0.1.3
```

Play video

```sh
$ airplayer play http://example.com/my-video.mp4
AirPlay: http://example.com/my-video.mp4
   Time: 00:00:06 |=====                                             | 13% Streaming
```
